{% trans "Tagesordnung" %}

  • ↑ {% trans "Nach oben" %}
Tagesordnung verwalten

Verwalten Sie Ihre Tagesordung komfortabel mit OpenSlides. Einträge lassen sich einfach per Drag&Drop verschieben. Eine hierarchische Liste mit Unterpunkten ist möglich.

Einträge projizieren

Jeder Tagesordnungseintrag lässt sich per Mausklick auf dem Projektor anzeigen. Egal in welcher Reihenfolge! Auch Rednerlisten lassen sich so projizieren.

Rednerliste

Jeder Tagesordnungseintrag enthält eine unabhängige Rednerliste, die von der Versammlungsleitung verwaltet wird. Teilnehmer können sich selbst auf diese Liste setzen.

Erledigt markieren

Abgeschlossene Tagesordnungspunkte können als erledigt markiert und anschließend zur besseren Übersicht ausgeblendet werden.

Zeitplanung

Schätzen Sie für jeden Tagesordnungseintrag eine Dauer und erhalten Sie dadurch einen prognostierten Endzeitpunkt der Veranstaltung.

Tagesordnung als PDF

Die vollständige Tagesordnung lässt sich auch als PDF anzeigen.





Anträge

  • ↑ {% trans "Nach oben" %}
Anträge papierlos verwalten

Importieren Sie Anträge aus einer CSV-Datei. Legen Sie die automatische Nummerierungsart fest und bestimmen Sie die Zugriffsrechte für Anträge: Welche Gruppen dürfen Anträge sehen, erstellen, verwalten und unterstützen?

HTML-Editor

Für einfache Formatierung von Antragstexten und -begründung ist ein HTML-Editor ("CKEditor") integriert. Die Formatierungen werden auch in der PDF-Fassung unterstützt.

Versionshistorie und Log

Änderungen eines Antrags können als neue Version gespeichert werden und lassen sich vergleichen. Jede Bearbeitungsaktion wird protokolliert und lässt sich zur besseren Rückverfolgung in OpenSlides einsehen.

Unterstützen von Anträgen

Sie können einstellen, ob Unterstützer nötig sind, um einen Antrag zuzulassen. Teilnehmer können dann direkt in OpenSlides Ihre Zustimmung für einen Antrag äußern.

Sachgebiete verwalten

Ordnen Sie Sachgebiete Ihren Anträgen zu und vergeben Sie zusätzlich ein passendes Prefix für die Antragsbezeichner, z.B. "SA 1".

Antragsbuch als PDF

Alle vorhandenen Anträge lassen sich übersichtlich in ein PDF erstellen; inkl. Informationen zu Antragsteller, Abstimmgungsergebnis und Begründung eines Antrags.

Anträge in die Tagesordnung

Jeder Antrag lässt sich als Referenz zusätzlich in die Tagesordnung einsortieren.

Anträge abstimmen

Nach Abstimmung eines Antrags im Plenum wird das ausgezählte Ergebnis in OpenSlides eingegeben (mit OpenSlides selbst wird nicht abgestimmt!). Dabei sind auch spezielle Werte für die Mehrheit der Stimmen, oder nicht erfasste Stimmen möglich. Anträge werden immer mit Ja-Nein-Enthaltung abgestimmt. Optional sind die abgegebenen Stimmen.

Arbeitsabläufe und Zustände definieren

Jeder Antrag durchläuft verschiedene Phasen in demokratischen Versammlungen. Nutzen Sie einen der zwei voreingestellten Arbeitsablä̈ufe mit vordefinierten Zuständen, die ein Antrag typischerweise einnehmen kann oder konfigurieren Sie Ihren individuellen Antragsablauf selbst. (Details im Handbuch.)




Wahlen

  • ↑ {% trans "Nach oben" %}
Wahlen verwalten

Jede Wahl besteht aus Name und Anzahl der zur Wahl stehenden Posten sowie - optional - aus einer Beschreibung und einem Kommentar für den Stimmzettel.

Stimmzettel als PDF

Geheime Wahlen benötgen Stimmzettel. Und diese können Sie schnell und unkompliziert direkt mit OpenSlides generieren - mit passenden Ankreuzfeldern vor jedem Kandidaten.

Unterstützte Wahlmethoden

OpenSlides unterstützt die Auswahl von 2 Wahlmethoden: "Eine Stimme pro Kandidat/in" oder "Ja, Nein, Enthaltungs pro Kandidat/in". Eine automatische Zuordnung beider Methoden, abhängig von der Anzahl der Kandidaten und zu vergebenen Posten, ist die Voreinstellung in OpenSlides. Ungültige und abgegebene Stimmen können zusätzlich immer erfasst werden.

Kandiatenliste aufstellen

Kandidaten können für eine Wahl direkt aus der Teilnehmerliste vorgeschlagen werden. Angemeldete Teilnehmer können per Mausklick sogar selbst kandidieren - sofern Sie die Berechtigung dazu haben.

Mehrere Wahlgänge

Mit OpenSlides können Sie beliebig viele Wahlgänge durchführen - und diese übersichtlich nebeneinander darstellen. Auch Stichwahlen sind leicht damit möglich.

Wahlergebnisse eingeben und darstellen

Sind die Wahlergebnisse ausgezählt, werden sie in OpenSlides vom Versammlungsleiter eingegeben. Per Knopfdruck werden die Zahlen auch für alle anderen Teilnehmer sichtbar.

Alle Wahlergebnisse in einem PDF

Exportieren Sie alle Wahlen ins PDF und geben Sie am Ende der Versammlung allen Delegierten eine Zusammenfassung aller gewählten Ämter. Oder nutzen Sie das PDF sogar als Anhang für Ihr Beschlussprotokoll.




Teilnehmer

  • ↑ {% trans "Nach oben" %}
Teilnehmer/innen verwalten

Nutzen Sie den CSV-Import zur Übernahme aus Ihrer Teilnehmerverwaltung. Dabei kann auch die Gliederungsebene, das Amt und ein Kommentar angegeben werden.

Gruppen und Rechte festlegen

Was jeder Teilnehmer sehen darf und was nicht, definieren Sie mit einzelnen Rechten. Mehrere Rechteeinstellungen können Sie in einer Gruppe zusammenfassen. Auch ein Gast (Anonymous-Nutzer) ist so aktivierbar.

Zugangszettel mit Erstpasswort und QR-Code

Verteilen Sie beim Empfang generierte Zettel mit Passwörtern und einem QR-Code zum Abscannen Ihrer OpenSlides-URL an Ihre Teilnehmer.




Allgemein

  • ↑ {% trans "Nach oben" %}
Portable Version für Windows

OpenSlides ist für Windows als portable Version verfügbar. Seit 1.4 mit grafischem Frontend und komfortabler Einrichtung einer automatischen Backupfunktion der SQLite-Datenbank.

Upload und Download von Dateien

Nutzen Sie während Ihrer Versammlung eine gemeinsame Dokumentenablage für alle Teilnehmer. Regeln Sie, welche Gruppen Upload- und Download-Rechte bekommen sollen.

Responsive Design mit Bootstrap

OpenSlides erscheint im modernen, responsiven Design auf Basis von Bootstrap und ist optimiert für unterschiedliche Mobilgeräte.

OpenSlides ist mehrsprachig

OpenSlides ist vollständig internationalisiert und ist derzeit in Deutsch, Englisch und Französisch verfügbar. (Weitere Sprachen sind gerne willkommen!)

Tornado Webserver und SQLite

OpenSlides integriert den leistungsfähigen, pythonbasierten Tornado Webserver. Als Datenbank nutzt OpenSlides voreingestellt SQLite.

Projektortemplate anpassbar

Die Projektoransicht ist leicht an Ihre individuellen Gestaltungsrichtlinien anpassbar; über eine zentral HTML- und CSS-Datei.
