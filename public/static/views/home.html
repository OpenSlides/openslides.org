What is OpenSlides?

OpenSlides helps to prepare and performe your assembly. You can manage agenda items, motions, elections, lists of speakers, documents and participants. Everything can be projected to the screen.

Features

Agenda

Be flexible: Add, modify or move agenda items during operation.

  • List of speakers for each agenda item.
  • Always have the current list of speakers visible (as overlay or on second projector).
  • Use countdowns to restrict speaking times.
  • Automatic numbering of items.
  • Scheduling: estimate duration of each agenda item.
  • Integrate unofficial items (e.g. breaks) into agenda.
  • Mark and hide closed agenda items.
  • Export agenda as PDF.
  • Control projector from sidebar.
  • Sort agenda by drag and drop.

Motions

Save paper: Submit and manage motions and voting results digitally.

  • Manage motions with line numbers.
  • Customizable motion states and workflows.
  • Manage categories, blocks and tags.
  • Define required supporters per motion.
  • Support amendments and change recommendations.
  • Show/hide each motion in agenda.
  • Version history to maintain modifications.
  • Inline editor to format motion text and reason.
  • Allow comment fields for motions.
  • Record voting results.
  • Export motions as PDF, CSV or DOCX.

Elections

Accelerate the procedure: Generate ad hoc ballots for your secret elections.

  • Nominate candidates from list of participants.
  • Print personalized ballot papers for secret elections.
  • Supported vote options: Yes/No/Abstain or one vote per candidate.
  • Calculated percentages on different 100% bases.
  • Several ballots.
  • Enter and project election results as diagram.
  • Export elections (with all results) as PDF.

More features

  • Create multiple projectors.
  • Manage participants, groups and permissions.
  • Document filing with PDF, image and video projection.
  • CSV import and export for agenda, motions and users.
  • Group chat.
  • Customizeable front page.
  • Full text search.
  • Customizeable projector template.
  • Responsive design.
  • Portable Version for Windows.
  • Multilingual (English, German, French, Czech, Portuguese, Spanish and Russian).
  • OpenSlides is a plattform independet web application based on Python 3 and AngularJS with a REST-API.
